# Building Sass

Install gulp

```sh
npm install -g gulp
```

Install npm packages

```sh
npm install
```
Install Browser Sync

```sh
npm install -g browser-sync
npm install browser-sync --save-dev
```

Compile Sass

```sh
npm sass
```

Watch sass and recompile

```sh
npm sass:watch
```
